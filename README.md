[Requirements]
- Python
- PIL / Pillow module

[How To Run]
1. Download the file (code -> download ZIP)
2. Extract it
3. Open your terminal / cmd
4. navigate to ImageToAscii.py path
5. use: 'py ImageToAscii.py' to run the script
6. write the path for the image you want to convert
